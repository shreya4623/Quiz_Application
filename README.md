# Quiz Application

## 📌 Project Overview

The Quiz Application is a console-based application developed using Python that enables users to participate in multiple-choice quizzes and evaluate their knowledge through an interactive question-and-answer format. The application presents questions with multiple options, validates user responses, calculates scores automatically, and provides detailed performance feedback at the end of the quiz.

The system is designed to make learning and self-assessment more engaging by displaying questions in a random order for each attempt. Users can answer questions by selecting the corresponding option number, and the application ensures valid input before proceeding. After completing the quiz, users receive their total score, percentage, pass/fail status, and a review of correct and incorrect answers.

## 🚀 Key Features

### Quiz Management

* Multiple-choice question format
* Four options for each question
* Randomized question order
* User-friendly console interface

### User Interaction

* Input validation for answer selection
* Easy navigation through questions
* Interactive quiz experience
* Real-time response collection

### Evaluation System

* Automatic score calculation
* Percentage computation
* Pass/Fail determination
* Performance feedback generation

### Result Analysis

* Display total score
* Show percentage obtained
* Review correct and incorrect answers
* Provide performance summary
